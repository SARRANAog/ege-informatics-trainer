# EGE Informatics Trainer

Офлайн-десктопное приложение для подготовки к ЕГЭ по информатике.

## Что внутри
- Python + pywebview для desktop shell
- FastAPI для локального backend API
- React + TypeScript + Vite + TailwindCSS для интерфейса
- SQLite для локального профиля и прогресса
- Monaco Editor для кодовых задач
- Структура под дорожку обучения, теорию, практику, weekly review и пробник

## Статус
Это **полный каркас проекта**: папки, стартовые файлы, базовые роуты, минимальные страницы и пример контента.
Это **не финально реализованный тренажёр со всем контентом 1–27**, а рабочая база для быстрой сборки.

## Быстрый запуск в режиме разработки

### Backend
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn backend.api.app:app --reload --host 127.0.0.1 --port 8765
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Desktop shell
Для локальной сборки интерфейс должен быть собран:
```bash
cd frontend
npm install
npm run build
cd ..
python desktop/main.py
```
